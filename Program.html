Introdução a Programação

O que é Programação?

Programação é o processo de criar instruções para que um computador execute tarefas. Essas instruções são escritas em uma linguagem de programação, como Python, JavaScript, C, Java, etc.

De forma simples: é como ensinar o computador a seguir um conjunto de passos.

Por que aprender Programação?

  • Resolver problemas de forma automática.
  • Criar sites, aplicativos, jogos e sistemas.
  • Trabalhar em áreas como ciência de dados, inteligência artificial, segurança digital (como o pentest que você mencionou antes).
  • Estimular o raciocínio lógico.

Conceitos básicos

  1. Algoritmo

    2. Uma sequência de passos para resolver um problema.

    Exemplo (fazer café)

                            1. Esquentar água
                        2. Colocar pó no filtro
                        3. Jogar água quente
                        4. Servir
